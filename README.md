# Hape Hub Backend

## Description

Hape Hub Backend adalah backend dari aplikasi Hape Hub yang dibuat menggunakan PHP.
 
## Features

- Login
- Register
- Get All Data Hape
- Get Data Hape By ID

## Tech 

- PHP

## Requirements

 - XAMPP 7.4 or later
 - PHP 8.1 or later
